# JAK Tech Firm Website (Static)

This is a clean, flyer-inspired, fully functional **static** website:
- Modern responsive design (mobile + desktop)
- WhatsApp booking form (opens WhatsApp with a prefilled message)
- Click-to-call phone links
- Instagram + Maps links
- No backend required

## Edit essentials
Open `script.js` and change:
- `WHATSAPP_NUMBER` (format: country code + number without leading 0)

Open `index.html` and change:
- Business hours (optional)
- Instagram handle, phone numbers, location text

## Run locally
Just double-click `index.html` OR use a local server:
- VS Code: install "Live Server" and click "Go Live"

## Deploy (easy)
Upload the 3 files to any hosting:
- Netlify / Vercel / GitHub Pages
- cPanel shared hosting
